# Lista de Tareas

Este proyecto es una aplicación de lista de tareas sencilla creada con HTML, CSS y JavaScript. La aplicación permite a los usuarios agregar tareas, marcarlas como completadas y eliminarlas.

## Características

- **Visualización de la fecha actual**.
- **Agregar tareas pendientes**.
- **Marcar tareas como completadas**.
- **Eliminar tareas**.

## Uso

1. **Agregar una tarea**: Escribe la tarea en el campo de texto y haz clic en el icono de agregar tarea (📋).
2. **Marcar una tarea como completada**: Haz clic en el icono de check (✔️) junto a la tarea.
3. **Eliminar una tarea**: Haz clic en el icono de la papelera (🗑️) junto a la tarea.
